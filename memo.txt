dotnet new console - создать новый проект
dotnet run         - зупустить проект

Синтаксис: начало

Write();           - Вывод в одну строку
WriteLine();       - В конце перейти на новую строку
ReadLine();        - Считать строку из терминала

Остаток обрезать с помощью строкового метода:
"{0: 0.00}" ... "{0: 0.0000000}"

Случайности:
new Random().Next(min, max)

даст случайное целое число от min до max-1:
[min, max-1] или [min, max]

% - деление с остатком

ToLower() позволяет учитывать разные буквы по велечине в слове
string string.ToLower() (+ 1 перегрузка)
